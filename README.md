## 📝 Descrição da Atividade
O objetivo deste trabalho foi realizar a montagem de um circuito utilizando o **Arduino Uno** e uma **placa de ensaio (protoboard)** para controlar componentes de saída.

Seguindo as orientações passadas pelo professor, o projeto foi estruturado com:
*   **02 LEDs:** Configurados para atuar de forma alternada/simultânea.
*   **Resistores:** Para proteção dos componentes.
*   **Lógica de Programação:** Implementada para gerenciar o comportamento das luzes conforme o roteiro da aula.

---

## 🛠️ Desenvolvimento
O projeto foi inteiramente simulado e testado no **Tinkercad**. Durante o processo, focamos em:
1.  **Organização da Protoboard:** Manter os jumpers e componentes bem dispostos para facilitar a leitura do circuito.
2.  **Resolução de Problemas:** Validar a conexão de terra (GND) comum para ambos os LEDs.

---

## 🚀 Conclusão
A atividade foi concluída com sucesso, apresentando o funcionamento pleno dos dois LEDs e demonstrando o domínio básico do grupo sobre as portas digitais do Arduino e o uso de protoboard.
